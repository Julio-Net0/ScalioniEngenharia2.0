Sobre este Projeto

A Residência Aurum é a síntese do luxo arquitetônico contemporâneo. Com 420m² de área construída, este projeto foi desenvolvido sob o conceito de "Maximalismo Minimalista", onde cada detalhe estrutural serve tanto à estética quanto à performance. Uma obra-prima da engenharia que redefine o morar de alto padrão.

architecture Especificações Técnicas

Área Total 420 m²
Número de Suítes 4 suítes / 1 master
Vagas na Garagem 3 vagas cobertas
Terreno Mínimo 20m x 25m / 500m²
Pavimentos 2 pavimentos
Padrão Alto Padrão NBR 12721
