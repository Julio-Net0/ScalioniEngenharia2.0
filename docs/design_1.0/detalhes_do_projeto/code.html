Sobre o Projeto

A Residência Scalioni é uma celebração da arquitetura contemporânea e do maximalismo elegante. Localizada em um terreno com declive acentuado, o projeto buscou integrar a topografia original com volumes horizontais que parecem flutuar sobre a paisagem mineira.

O uso de materiais nobres como o mármore travertino bruto, madeira nogueira e estruturas metálicas aparentes cria um contraste sofisticado. O interior foi planejado para maximizar a entrada de luz natural, utilizando pé-direito duplo nas áreas sociais e grandes panos de vidro que eliminam as barreiras entre interior e exterior.
